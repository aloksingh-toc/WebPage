# NewProject

Static marketing website for Mangalam HDPE Pipes.

## Overview

This project is a lightweight front-end site built with plain HTML, CSS, and JavaScript. It includes:

- responsive landing page sections
- product carousel with zoom preview
- sticky header and mobile navigation
- accessible product submenu improvements
- FAQ accordion and modal forms

## Project Structure

- `index.html` - page structure and content
- `styles.css` - layout, components, and responsive styling
- `script.js` - interactive behaviors
- `*.png` - website image assets

## Running Locally

Open `index.html` directly in a browser, or serve the folder with any static file server.

## Notes

- The repo currently tracks the image assets used by the page.
- `Mangalam_HDPE_Website.zip` is kept out of version control.
